SPONGEY CLUB AUTOMATIC MEMBER POSTER

1. Upload the CONTENTS of this folder to Netlify, including:
   index.html
   member-poster.png
   netlify.toml
   netlify/functions/member-count.mjs

2. After deployment, open:
   https://YOUR-SITE.netlify.app/api/member-count

   It should display JSON containing a count number.

3. Open your main Netlify site. The poster starts at 22 and automatically replaces it with the live Spongey Club member count.

4. The count refreshes every 5 minutes and when the page becomes visible again.
